4 Min/Max Width

min-w-full = 100%
min-w-min = min-content
min-w-max = max-content

Alternatif untuk gap pada elemen non-flex/grid.

space-y-4

üí° Tip: Untuk Flexbox dan Grid, lebih baik gunakan gap-* karena tidak menambahkan margin pada item pertama/terakhir.

‚ö†Ô∏è Kesalahan Umum Pemula

  • ‚Ä¢ Menggunakan margin untuk spacing antar item dalam flex/grid (gunakan gap)
  • ‚Ä¢ Lupa bahwa h-full butuh parent dengan height yang didefinisikan
  • ‚Ä¢ Menggunakan w-screen di dalam container bisa menyebabkan horizontal scroll
  • ‚Ä¢ Tidak menggunakan max-w-* pada container untuk readability

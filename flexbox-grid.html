üèóÔ∏è Struktur Layout

Flexbox digunakan untuk:

  • ‚Ä¢ flex - Container utama (sidebar + content)
  • ‚Ä¢ flex flex-col - Sidebar & main content area
  • ‚Ä¢ justify-between - Header layout
  • ‚Ä¢ flex-shrink-0 - Sidebar tidak menyusut
  • ‚Ä¢ flex-1 - Konten mengisi sisa ruang

Grid digunakan untuk:

  • ‚Ä¢ grid grid-cols-3 - Stats cards
  • ‚Ä¢ lg:col-span-2 - Chart lebih lebar
  • ‚Ä¢ gap-3 - Spacing konsisten
  • ‚Ä¢ grid-cols-1 md:grid-cols-2 lg:grid-cols-3 - Responsif

üíª Kode Dashboard Layout

dashboard-layout.html 
<!-- Dashboard Container -->
<div class="flex h-screen">
  
  <!-- Sidebar (Flexbox) -->
  <aside class="w-64 bg-gray-800 flex-shrink-0 flex flex-col">
    <div class="p-4 border-b border-gray-700">Logo</div>
    <nav class="flex-1 p-4">Menu items</nav>
  </aside>
  
  <!-- Main Area (Flex column) -->
  <div class="flex-1 flex flex-col">
    
    <!-- Header -->
    <header class="flex items-center justify-between px-6 py-4 border-b">
      <h1>Dashboard</h1>
      <div class="flex items-center gap-4">Profile</div>
    </header>
    
    <!-- Content (Grid) -->
    <main class="flex-1 p-6 overflow-y-auto">
      <div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
        <div class="p-4 rounded-lg bg-gray-800">Card</div>
      </div>
    </main>
  </div>
</div>

‚ú® Tantangan Tambahan:

  • ‚ñ° Tambahkan sidebar yang bisa collapse dengan w-16 ‚Üí w-64
  • ‚ñ° Buat responsive: hide sidebar di mobile dengan hidden lg:flex
  • ‚ñ° Tambahkan grid grid-rows-[auto_1fr_auto] untuk header-content-footer
  • ‚ñ° Coba gunakan order untuk swap posisi di mobile

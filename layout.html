10 Position

static

Default - mengikuti normal flow

relative

Relatif dari posisi normal, jadi anchor untuk absolute

absolute

Relatif dari parent non-static terdekat

fixed

Relatif dari viewport, tetap saat scroll

sticky

Kombinasi relative + fixed - menempel di posisi tertentu saat scroll

Contoh Sticky Navbar:

<nav class="sticky top-0 bg-white shadow-md z-50">
  <!-- Navbar akan menempel di atas saat scroll -->
</nav>

üí° Perbedaan hidden vs invisible:
hidden = display:none (tidak ada di layout)

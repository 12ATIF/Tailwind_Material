1 Apa itu Tailwind CSS?

Tailwind CSS adalah framework CSS yang menggunakan pendekatan utility-first. Artinya, kamu membangun desain langsung di HTML menggunakan class-class kecil yang sudah tersedia.

üí° Analogi: Bayangkan kamu punya kotak LEGO dengan banyak potongan kecil. Tailwind CSS seperti LEGO - kamu menggabungkan class-class kecil untuk membuat desain yang besar.

Perbandingan: CSS Tradisional vs Tailwind

CSS Tradisional ‚ùå
/* styles.css */
.btn-primary {
  background-color: #3B82F6;
  color: white;
  padding: 0.5rem 1rem;
  border-radius: 0.5rem;
  font-weight: 600;
}

.btn-primary:hover {
  background-color: #2563EB;
}
Tailwind CSS ‚úÖ
<!-- Langsung di HTML -->
<button class="bg-blue-500 
  text-white 
  px-4 py-2 
  rounded-lg 
  font-semibold 
  hover:bg-blue-600">
  Klik Saya
</button>

2 Kelebihan & Kekurangan

‚úÖ Kelebihan

  • Cepat: Tidak perlu berpindah-pindah file CSS
  • Konsisten: Design system bawaan yang rapi
  • Responsive: Mudah membuat layout responsive
  • File kecil: Production build sangat optimal

‚ùå Kekurangan

  • HTML panjang: Banyak class di elemen
  • Learning curve: Perlu hafal class-class
  • Setup: Butuh build process untuk production

3 Kapan Cocok Menggunakan Tailwind?

‚úÖ Cocok untuk:

  • Proyek baru yang butuh development cepat
  • Aplikasi dengan komponen yang sering dipakai ulang
  • Tim yang ingin design system konsisten
  • Prototype dan MVP

‚ùå Mungkin kurang cocok untuk:

  • Proyek yang sudah punya CSS besar dan established
  • Tim yang tidak familiar dengan utility-first approach
  • Proyek sangat kecil (satu halaman statis)

4 Installation

Ada beberapa cara menginstall Tailwind CSS. Untuk belajar, kita pakai cara paling mudah: CDN.

RECOMMENDED Cara 1: Menggunakan CDN (Paling Mudah)

Tambahkan script ini di dalam tag <head>:

index.html 
<script src="https://cdn.tailwindcss.com"></script>
‚ö†Ô∏è Catatan

CDN cocok untuk belajar dan prototyping. Untuk production, gunakan build process dengan npm/yarn.

üìÑ Contoh File HTML Lengkap

üìã Penjelasan Class yang Digunakan:

Class Fungsi
bg-gray-100 Background warna abu-abu terang
min-h-screen Minimum height 100% viewport
container mx-auto Container centered dengan max-width
p-8 Padding 2rem (32px) semua sisi
text-4xl font-bold Teks besar (2.25rem) dan tebal
text-blue-600 Warna teks biru
mt-4 Margin top 1rem (16px)
hover:bg-blue-600 Background biru lebih gelap saat hover
üí° Tips

Coba ubah nilai class seperti text-4xl menjadi text-6xl atau bg-blue-500 menjadi bg-green-500 untuk melihat perubahannya!

1.2 Boilerplate HTML dengan Tailwind CDN

Kita akan menggunakan Tailwind CSS via CDN karena ini cara tercepat untuk prototyping. Berikut template dasar yang digunakan:

index.html 
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Busana Muslim Nusantara</title>
    
    <!-- Tailwind CSS CDN -->
    <script src="https://cdn.tailwindcss.com"></script>
    
    <!-- Google Fonts -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Playfair+Display:wght@400;500;600;700&display=swap" rel="stylesheet">
    
    <!-- Custom CSS -->
    <link rel="stylesheet" href="assets/css/style.css">
</head>
<body class="font-inter bg-background">

    <!-- Konten website akan di sini -->

    <script src="assets/js/main.js"></script>
</body>
</html>

<script src="https://cdn.tailwindcss.com"> adalah cara tercepat untuk menggunakan Tailwind. Untuk production, disarankan install via npm untuk optimasi file size.

1.3 Custom Tailwind Configuration

Project ini menggunakan custom colors dan fonts. Konfigurasi dilakukan langsung di HTML dengan script:

Tailwind Config (dalam <head>) 
<script>
    tailwind.config = {
        theme: {
            extend: {
                // Custom Colors - Nuansa Olive/Sage
                colors: {
                    primary: '#5C6E4E',        // Olive hijau utama
                    'primary-dark': '#4A5A3E', // Untuk hover state
                    'primary-light': '#7A8F6A',
                    secondary: '#F5F0E6',      // Krem lembut
                    accent: '#8B7355',         // Cokelat hangat
                    'accent-dark': '#6B5A45',
                    background: '#FAFAF7',     // Off-white
                    'custom-text': '#3D3D3D',  // Abu gelap
                    'text-light': '#6B6B6B',
                    'custom-border': '#E5E0D5',
                    gold: '#C4A962',           // Untuk rating
                },
                // Custom Fonts
                fontFamily: {
                    'playfair': ['Playfair Display', 'serif'],
                    'inter': ['Inter', 'sans-serif'],
                }
            }
        }
    }
</script>

Warna Olive/Sage green dipilih karena memberikan kesan natural, elegan, dan syar'i - sangat cocok untuk brand busana muslim!

1.4 Typography: Font Pairing

Project ini menggunakan kombinasi 2 font yang saling melengkapi:

Font Tipe Penggunaan Class Tailwind
Playfair Display Serif Heading, Logo, Judul font-playfair
Inter Sans-serif Body text, Deskripsi font-inter

1.5 Menggunakan Class pada Body

Perhatikan class yang digunakan pada <body>:

Body Tag 
<body class="font-inter bg-background">
Class Fungsi
font-inter Set default font seluruh halaman ke Inter (dari custom config)
bg-background Background off-white #FAFAF7 (dari custom config)

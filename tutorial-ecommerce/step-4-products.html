4.4 Custom CSS untuk Card

style.css 
/* Product Card Container */
.product-card {
    background-color: white;
    border-radius: 1rem;
    overflow: hidden;
    transition: all 0.3s ease;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.05);
}

/* Hover: Float + Shadow */
.product-card:hover {
    transform: translateY(-8px);
    box-shadow: 0 12px 24px rgba(0, 0, 0, 0.1);
}

/* Image Zoom on Hover */
.product-card img {
    transition: transform 0.5s ease;
}

.product-card:hover img {
    transform: scale(1.05);
}

/* Badge Styles */
.badge {
    display: inline-flex;
    padding: 0.25rem 0.75rem;
    border-radius: 9999px;
    font-size: 0.75rem;
    font-weight: 600;
    text-transform: uppercase;
}

.badge-sale { background: #DC2626; color: white; }
.badge-new { background: #C4A962; color: white; }
.badge-bestseller { background: #5C6E4E; color: white; }

/* Star Rating */
.star-rating {
    color: #C4A962; /* Gold color */
}
üí° Transform + Shadow = Premium Feel

Kombinasi translateY(-8px) + box-shadow yang lebih besar saat hover menciptakan efek "floating" yang terasa premium. User akan merasa card "terangkat" saat di-hover!

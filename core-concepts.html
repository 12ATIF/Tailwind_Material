2 Hover, Focus, dan States Lainnya

Tailwind memungkinkan kita menambahkan style untuk berbagai state dengan mudah menggunakan prefix.

hover:

Ketika mouse di atas elemen

focus:

Ketika elemen difokuskan

active:

Ketika elemen diklik

disabled:

Ketika elemen disabled

Preview (arahkan mouse ke button)
hover-button.html

  text-white px-4 py-2 rounded-lg
  transition-colors duration-200">
  Hover Saya
</button>

3 Responsive Design

Tailwind menggunakan pendekatan mobile-first. Class tanpa prefix berlaku untuk semua ukuran, lalu kita tambahkan prefix untuk layar yang lebih besar.

Prefix Minimum Width CSS
sm: 640px @media (min-width: 640px)
md: 768px @media (min-width: 768px)
lg: 1024px @media (min-width: 1024px)
xl: 1280px @media (min-width: 1280px)
2xl: 1536px @media (min-width: 1536px)
responsive-grid.html 
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
Class Fungsi
grid-cols-1 1 kolom (default di mobile)
md:grid-cols-2 2 kolom di tablet (â‰¥768px)
lg:grid-cols-3 3 kolom di desktop (â‰¥1024px)
gap-4 Jarak antar item 1rem

6 Menambahkan Custom Styles

Ketika class bawaan tidak cukup, kamu bisa menggunakan arbitrary values dengan tanda kurung siku.

arbitrary-values.html 
<!-- Ukuran custom -->
<div class="w-[137px] h-[68px]"></div>

<!-- Warna custom -->
<div class="bg-[#1da1f2] text-[rgb(255,255,255)]"></div>

<!-- Font custom -->
<p class="text-[22px] leading-[32px]"></p>
âš ï¸ Catatan

Gunakan arbitrary values hanya jika class standar Tailwind tidak cukup. Terlalu banyak arbitrary values bisa membuat kode sulit dipelihara.

  • â€¢Terlalu banyak arbitrary values: Gunakan hanya jika benar-benar perlu

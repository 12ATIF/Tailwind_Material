2.4 Custom CSS untuk Navbar

Beberapa styling tidak bisa dilakukan dengan Tailwind saja, kita perlu custom CSS:

style.css 
/* Glassmorphism effect */
.navbar {
    backdrop-filter: blur(10px);
    background-color: rgba(255, 255, 255, 0.95);
    transition: all 0.3s ease;
}

/* Shadow saat scroll (via JavaScript) */
.navbar.scrolled {
    box-shadow: 0 2px 20px rgba(0, 0, 0, 0.1);
}

/* Nav link dengan animated underline */
.nav-link {
    position: relative;
    color: var(--color-text);
    font-weight: 500;
    padding: 0.5rem 0;
    transition: color 0.3s ease;
}

.nav-link::after {
    content: '';
    position: absolute;
    bottom: 0;
    left: 0;
    width: 0;
    height: 2px;
    background-color: var(--color-primary);
    transition: width 0.3s ease;
}

.nav-link:hover::after,
.nav-link.active::after {
    width: 100%;
}

/* Cart Badge */
.cart-badge {
    position: absolute;
    top: -8px;
    right: -8px;
    width: 20px;
    height: 20px;
    background-color: var(--color-accent);
    color: white;
    border-radius: 50%;
    display: flex;
    align-items: center;
    justify-content: center;
    font-size: 0.75rem;
    font-weight: 600;
}
üí° Glassmorphism

backdrop-filter: blur(10px) + background semi-transparan = efek kaca buram yang modern. Konten di belakang navbar akan terlihat blur!
